validationColors(val, regEx, obj, mode, locale)

Sets the form field color to red or green depending on the contents
Parameters:
Name Type Description
val string | integer | float Value of a specific form field
regEx regex Regular Expression string to compare value with
obj object Form field to change color of
mode integer Can be 0 or 1. In Mode 0, regex is not used...instead you can define whether to set the field to red or green by sending a boolean through the regEx parameter
locale integer Can be 0 or 1. 0 sets the form fields parent element to a color while 1 sets the form field itself to a color.
